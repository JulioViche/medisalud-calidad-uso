# Validación end-to-end

La validación manual cubre:

1. Arranque saludable de todos los servicios con Docker Compose en WSL.
2. Portal React conectado al Gateway.
3. Navegación por roles y ejecución de HCE lenta.
4. Persistencia en PostgreSQL y SQL Server.
5. Evento HCE publicado en RabbitMQ.
6. Flutter web en viewport móvil consumiendo el mismo Gateway.
7. Informe LaTeX renderizado y revisado página por página.
